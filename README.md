# CSS Clip-Path Examples Showcase

A comprehensive collection of 40 interactive CSS `clip-path` examples, demonstrating everything from basic shapes to advanced animations and interactions. Built with Next.js and organized into three difficulty levels to help developers learn and master CSS clipping techniques.

## Overview

CSS `clip-path` is a powerful property that allows you to clip elements to basic shapes, custom polygons, or SVG paths. This repository serves as a learning resource and reference guide for developers who want to understand and implement various clipping techniques in their projects.

### What You'll Learn

- Basic shape clipping (circles, ellipses, polygons)
- Advanced animations and transitions
- Interactive hover effects and reveals
- Scroll-triggered animations
- Complex SVG path morphing
- 3D perspective effects
- Multi-layer clipping techniques
- And much more!

## Getting Started

### Prerequisites

- Node.js 18+ installed
- pnpm package manager (recommended)

### Installation

1. Clone the repository:

  ```bash
  git clone <repository-url>
  cd clip-path-nextjs-experiments
  ```

2. Install dependencies:

  ```bash
  pnpm install
  ```

3. Run the development server:

  ```bash
  pnpm dev
  ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

  ```bash
  pnpm build
  pnpm start
  ```
